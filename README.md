# Set
A simple set implementation for go.

- [X] Safe for concurrent access.
- [x] Extensible by gene code
- [x] Type-safe
    - [x] int8
    - [x] int16
    - [x] int32
    - [x] int
    - [x] int64
    - [x] uint8
    - [x] uint16
    - [x] uint32
    - [x] uint
    - [x] uint64
    - [x] float32
    - [x] float64
    - [x] string
    - [x] time.Time (round to second)
